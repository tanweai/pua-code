# Pua Code

![](https://img.shields.io/badge/Node.js-18%2B-brightgreen?style=flat-square) [![npm]](https://www.npmjs.com/package/@pua-ai/pua-code)

[npm]: https://img.shields.io/npm/v/@pua-ai/pua-code.svg?style=flat-square

Pua Code is an agentic coding tool that lives in your terminal, understands your codebase, and helps you code faster by executing routine tasks, explaining complex code, and handling git workflows -- all through natural language commands. Use it in your terminal, IDE, or tag @pua on Github.

**Learn more at [Pua Code Homepage](https://pua.com/product/pua-code)** | [Documentation](https://code.pua.com/docs/en/overview)

<img src="https://github.com/puas/pua-code/blob/main/demo.gif?raw=1" />

## Get started

1. Install Pua Code:

```sh
npm install -g @pua-ai/pua-code
```

2. Navigate to your project directory and run `pua`.

## Reporting Bugs

We welcome your feedback. Use the `/bug` command to report issues directly within Pua Code, or file a [GitHub issue](https://github.com/puas/pua-code/issues).
